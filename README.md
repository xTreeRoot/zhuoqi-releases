# Zhuoqi Releases

`桌栖 · 璃夏` 是一个把 `AI 聊天`、`角色陪伴`、`声音能力`、`微信 / QQ 本地整理`、`提炼沉淀` 和 `自动化扩展` 放到同一个桌面工作区里的 macOS 应用。

它不是单纯的聊天窗口，而是一个能接住本机聊天材料、继续加工成草稿、扩展和工作区内容的桌面 AI 容器。

这个仓库只用于存放 `Zhuoqi` 的正式发布包、安装包和版本文件，不放源代码。

## 页面与功能详解

### 关于

- 能做什么：查看产品定位、账号状态、套餐价格、权益说明、功能手册和常见问题。
- 免费可做：完整查看当前版本说明、价格和上手信息。
- SVIP 多什么：不会额外开放新按钮，主要是页面里展示的账号状态和已开通权益会更多。
- 适合谁用：第一次接触软件、想先判断“哪些免费、哪些付费”的用户。

### 聊天

- 能做什么：和 AI 持续对话，自动接回当前聊天空间，保留上下文，支持引用消息继续聊。
- 可以完成什么：问答、陪伴聊天、写作协作、灵感记录、把微信 / QQ / 提炼结果继续接到当前会话里处理。
- 免费可做：正常聊天、发送文本、引用消息、清空当前消息区、接收语音转写后的文本输入。
- SVIP 多什么：聊天页本身不是 SVIP 页；但如果你在声音页开通了高级语音能力，就能把语音输入和远端音色能力接入聊天页一起用。
- 本地前置条件：本机需要有可访问的聊天网关，默认地址是 `ws://127.0.0.1:18789`。如果你本地没有运行 `OpenClaw / 龙虾` 这类聊天服务，就需要在配置页改成自己的可用地址。
- 远端前置条件：如果聊天网关不是本机，而是远端服务，就要保证该地址当前可连接、当前聊天空间可用。

### 形象

- 能做什么：切换角色模型、切换渲染方式、查看模型状态、打开独立角色窗口。
- 可以完成什么：把角色作为桌面陪伴窗口长期悬挂，或者在不同模型之间快速切换展示效果。
- 免费可做：当前形象页的主要能力已经开放，包括切换模型和打开独立窗口。
- SVIP 多什么：当前实现里没有把形象页主流程再额外按 SVIP 拦住。
- 本地前置条件：本机需要存在可读取的 Live2D 模型目录，默认会读取项目的 `models/`，也可以在配置页改成自己的模型目录。
- 远端前置条件：无。

### 声音

- 能做什么：查看默认音色、浏览音色列表、搜索筛选、试听、切换默认音色、管理播报和语音输入设置、创建在线音色、保存声音档案。
- 可以完成什么：把桌面角色接入播报能力，做本机陪伴语音，或者做在线音色体验和语音输入。
- 免费可做：
  - 浏览全部声音页面内容
  - 刷新音色库
  - 试听本机免费音色
  - 把本机免费音色设为默认音色
  - 在不开启语音输入时保存基础播报设置
- SVIP 多什么：
  - 使用在线官方音色
  - 使用在线自定义音色
  - 保存和删除声音档案
  - 试听草稿音色
  - 开启语音输入
  - 使用完整的远端播报链路
- 本地前置条件：
  - 基础体验依赖 macOS 本机 `say` 音色
  - 语音输入还需要本机麦克风可用
  - 如果要创建在线自定义音色，需要准备参考音频
- 远端前置条件：
  - 在线音色、在线播报、草稿试听、语音输入依赖百炼兼容接口
  - 关键配置项是 `qwen.apiKey` 和 `qwen.baseUrl`

### 微信

- 能做什么：检查微信运行状态、授权状态、聊天读取状态、数据源状态，刷新会话列表，搜索联系人或群聊，浏览摘要，预览消息，整理并发送到当前聊天空间。
- 可以完成什么：把本机微信聊天变成 AI 可继续处理的上下文材料，而不是手动复制聊天记录。
- 免费可做：
  - 检查微信环境
  - 刷新状态和聊天列表
  - 搜索联系人或群聊
  - 浏览会话摘要
  - 开启自动更新
- SVIP 多什么：
  - 授权微信
  - 开启聊天读取
  - 打开聊天库
  - 打开录屏权限入口
  - 进入聊天内容预览
  - 查看附件、最近消息和时间范围内消息
  - 批量勾选聊天
  - 整理并发送到当前聊天空间
- 本地前置条件：
  - 本机已安装并登录微信
  - 高级整理前通常要完成临时签名、提取密钥和聊天读取授权
  - 首次使用时可能需要处理 macOS 的屏幕录制权限
- 远端前置条件：
  - 如果要把整理结果送回当前 AI 会话，需要聊天网关本身可访问

### QQ

- 能做什么：检查 QQ 运行状态、账号识别情况、聊天读取状态、刷新好友和群聊列表、搜索会话、浏览摘要、预览聊天并整理发送。
- 可以完成什么：把 QQ 沟通内容补进当前工作上下文，用于继续总结、推进和归档。
- 免费可做：
  - 检查 QQ 环境
  - 刷新状态和列表
  - 搜索好友或群聊
  - 浏览摘要
  - 开启自动更新
- SVIP 多什么：
  - 进入聊天内容预览
  - 查看最近聊天
  - 批量勾选会话
  - 整理并发送到当前聊天空间
- 本地前置条件：
  - 本机已安装并登录 QQ
  - 要真正读聊天，必须同时满足本地聊天库已发现、数据库密钥已拿到、读取器已就绪
- 远端前置条件：
  - 如果要把整理结果送回当前 AI 会话，聊天网关需要可访问

### 提炼

- 能做什么：从微信或 QQ 聊天里生成结构化提炼稿，并输出为 `Skill / 蒸馏包` 或 `Claw / OpenClaw 文件`。
- 可以完成什么：
  - 做总结
  - 做待办提取
  - 做人物关系梳理
  - 把聊天材料沉淀成扩展或工作区文件
- 当前支持的提炼控制项：
  - 选择微信或 QQ 作为来源
  - 选择时间窗口
  - 选择人物关系视角，例如自己、同事、导师、亲人、伴侣、朋友、公众人物
  - 选择增强提炼通道或 OpenClaw 通道
  - 保存到扩展目录或应用到 OpenClaw 工作区
- 免费可做：当前不能直接进入这个页面。
- SVIP 多什么：开放整页提炼流程和结果落地能力。
- 本地前置条件：
  - 微信或 QQ 的聊天源本身已经可读
  - 如果要保存成扩展，需要可写扩展目录
  - 如果要应用到工作区，需要可写 OpenClaw 工作区
- 远端前置条件：
  - `增强提炼通道` 依赖 `qwen.apiKey`、`qwen.baseUrl`、`qwen.chatExtractionModel`
  - `OpenClaw 通道` 依赖当前聊天网关在线、当前会话能正常回复

### 配置

- 能做什么：维护外观、连接方式、角色与模型目录、声音默认值、项目目录、扩展目录、接口配置、更新设置和实际解析路径。
- 可以完成什么：
  - 修改主题、菜单停靠位置和关闭行为

  - 配置启动时是否自动拉取微信 / QQ 最新消息
  - 调整模型目录、声音默认值和扩展目录
  - 填写百炼兼容接口配置
  - 检查版本更新并下载安装包

- SVIP 多什么：配置页本身不额外收费，但其中填写的远端接口会决定高级功能能不能真正跑起来。
- 本地前置条件：部分目录和路径修改后，需要保存并重启软件。
- 远端前置条件：接口管理里填的服务地址和 Key 必须真实可用。

## 本地前置条件汇总

如果想把这些功能真正跑起来，至少要满足下面这些本地条件：

- 当前主要支持 `macOS`
- 需要有可访问的 `OpenClaw / 龙虾` 聊天网关，默认地址是 `ws://127.0.0.1:18789`
- Live2D 形象需要本地模型目录完整可读
- 本机免费音色依赖 macOS 系统 `say` 音色
- 语音输入需要本机麦克风
- 微信高级整理需要本机微信、录屏权限、签名与密钥读取链路
- QQ 高级整理需要本机 QQ、本地聊天库、数据库密钥和读取器
- 提炼落地到工作区或扩展时，需要本机存在可写目录

## 远端前置条件汇总

以下功能会依赖远端接口或远端服务：

- `聊天`：依赖聊天网关可连接
- `在线音色`：依赖百炼兼容接口
- `远端播报`：依赖百炼兼容接口
- `语音输入`：依赖百炼兼容接口和 API Key
- `增强提炼通道`：依赖百炼兼容接口和提炼模型

当前远端配置主要集中在：

- `qwen.apiKey`
- `qwen.baseUrl`
- `qwen.chatExtractionModel`

如果这些配置没填好，软件仍然可以保留大量本地免费能力，但在线音色、远端播报、语音输入和增强提炼不会正常工作。

## 这些功能最后能做成什么

把这些页面串起来之后，当前软件可以完成几类典型链路：

- 持续聊天：把角色、聊天空间和语音能力组合成一个长期运行的桌面 AI 工作台。
- 本地整理：把微信 / QQ 本地聊天整理成 AI 可继续处理的上下文，而不是手动复制聊天记录。
- 深度沉淀：把聊天继续提炼成 `Skill / 蒸馏包` 或 `Claw / OpenClaw 文件`，用于后续复用。
- 自动复用：把常用文案存进文案库，再通过自动化周期性触发。
- 桌面陪伴：把 Live2D 形象、本机免费音色或在线音色组合成角色陪伴体验。
